# Fantasy Tower Defense

A browser-based fantasy tower defense game built with Next.js and React.

## Requirements

- Node.js 20+
- Corepack enabled (`corepack enable`)

## Getting Started

Install dependencies:

```bash
corepack pnpm install
```

Run in development mode:

```bash
corepack pnpm dev
```

Open http://localhost:3000 in your browser.

## Available Scripts

- `corepack pnpm dev` — start local development server
- `corepack pnpm build` — create production build
- `corepack pnpm start` — run production server
- `corepack pnpm check` — run production build plus type/lint checks
- `corepack pnpm format` — format project files with Prettier
